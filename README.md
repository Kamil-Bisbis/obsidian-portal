# Portal

Portal creates editable embeds in Obsidian backed by real Markdown files.

A portal is a small window into one shared file. Insert the same portal in multiple notes, edit it from any place it appears.

## What it does

- Creates editable portal blocks using fenced code blocks.
- Stores portal content as normal Markdown files in `portals/`.
- Lets you edit portal content directly inside the note where it appears.
- Keeps multiple instances of the same portal connected through the same backing file.
- Supports creating a portal from selected text, the current line, or an empty starting point.
- Supports inserting an existing portal.
- Supports inserting the most recently edited portal.
